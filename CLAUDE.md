# CLAUDE.md / Project Conventions

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript (ES6+)
- Environment: Node.js (LTS) for tooling and dev server

## Build & Development Commands

- Local Development Server: `npm run dev`
- Run Tests: `npm test`

## Code Style & Guidelines

- Use clear, descriptive names for functions and variables.
- Write modular, single-purpose components.
- Follow Conventional Commits format for Git messages.
- Prioritize clean layout structures with semantic HTML elements.
